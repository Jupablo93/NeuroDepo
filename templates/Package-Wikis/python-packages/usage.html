Usage Examples

Visualizing a 3D Brain Volume

This example shows how to load and display a structural MRI scan using the plot_anat function.

from nilearn import plotting, datasets

mni152 = datasets.load_mni152_template()
plotting.plot_anat(mni152)

Displaying Functional MRI Activation

This snippet overlays statistical activation maps on the anatomical template.

from nilearn import plotting, image

stat_map = image.load_img('data/statistical_map.nii.gz')
plotting.plot_stat_map(stat_map, threshold=3.0)
